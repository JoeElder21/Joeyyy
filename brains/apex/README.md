# APEX Brain Specialist Unit

APEX is the professional, firm, project, technical, revenue, and career command center. Its five specialists form one operating unit under Agent 007:

1. **APEX War Architect** sets direction and operating campaigns.
2. **APEX Deal Engine** advances ethical, authorized opportunities.
3. **APEX Delivery Commander** drives committed work to verified completion.
4. **APEX Intelligence Forge** normalizes evidence and creates decision clarity.
5. **APEX Systems Blacksmith** converts proven repetition into durable infrastructure.

The operating loop is:

`evidence → intelligence → direction → opportunity or delivery → systemization → verified result`

## Ownership

- These agents may use only APEX-owned evidence.
- They may ask, challenge, and hand off only within the APEX unit.
- Agent 007 is the only cross-brain coordinator.
- Each agent has a unique logical memory namespace and exact proposed-write target array in [agents.toml](agents.toml).
- The specialists are read-only in shadow stage. Agent 007 alone holds the writer lease and verifies every mutation.

## File placement

Codex discovers project agents from the flat `.codex/agents/` directory, so native runtime definitions live there. This directory is the brain-owned source of roster metadata, memory boundaries, routes, outputs, and lifecycle state.

The repository is public. Store only sanitized contracts, tests, schemas, migration records, and synthetic examples. Live employer, client, project, opportunity, contact, quantity, schedule, and connector data remains in authorized APEX systems at runtime.
