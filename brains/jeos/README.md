# JEOS Brain Specialist Unit

JEOS is Joe's personal, private, life, faith, relationships, health-and-energy, home, and personal-administration command center. Its five specialists form one operating unit under Agent 007:

1. **JEOS Life Architect** sets personal direction and operating structure.
2. **JEOS Momentum Engine** turns intentions into starts, habits, and completion.
3. **JEOS Energy Director** protects sustainable capacity and recovery.
4. **JEOS Reflection Forge** turns lived evidence into lessons and experiments.
5. **JEOS Lifestyle Systems Builder** removes recurring personal-administration load.

The operating loop is:

`lived evidence → reflection → direction → momentum within capacity → systemization → verified result`

## Ownership

- These agents may use only JEOS-owned evidence.
- They may ask, challenge, and hand off only within the JEOS unit.
- Agent 007 is the only cross-brain coordinator.
- Each agent has a unique logical memory namespace and exact proposed-write target array in [agents.toml](agents.toml).
- The specialists are read-only in shadow stage. Agent 007 alone holds the writer lease and verifies every mutation.

## File placement

Codex discovers project agents from the flat `.codex/agents/` directory, so native runtime definitions live there. This directory is the brain-owned source of roster metadata, memory boundaries, routes, outputs, and lifecycle state.

The repository is public. Store only sanitized contracts, tests, schemas, migration records, and synthetic examples. Live journals, relationships, health, finance, addresses, schedules, personal messages, and connector data remain in authorized JEOS systems at runtime.
