## Decision

Replace the first shadow specialist corps with five mirrored functional classes deployed as ten separate agents:

| Class | APEX | JEOS |
|---|---|---|
| Strategy | `apex_war_architect` | `jeos_life_architect` |
| Opportunity / momentum | `apex_deal_engine` | `jeos_momentum_engine` |
| Execution / capacity | `apex_delivery_commander` | `jeos_energy_director` |
| Intelligence / reflection | `apex_intelligence_forge` | `jeos_reflection_forge` |
| Systems / automation | `apex_systems_blacksmith` | `jeos_lifestyle_systems_builder` |

Mirroring provides a consistent operating grammar. It does not create shared identity, memory, source access, write targets, roundtables, prompts, or direct communication.
